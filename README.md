# iPOS Inventory Audit

> 📂 **[VIEW PROJECT PRESENTATION (PDF)](presentation/Inventory_Audit_Full_Pipeline.pdf)**

## 🎯 Overview
This project automates the extraction and transformation of inventory data from the **iPOS (IVT)** system. It replaces **2+ hours of manual work per week** with a precise, automated pipeline for financial reconciliation and operational audit.

## 🛠 Tech Stack
* **Language:** Python 3.x.
* **Automation:** Selenium WebDriver (Auto-login & Token extraction).
* **Data Science:** Pandas & NumPy (High-speed data processing).
* **Cloud:** Google Sheets API (Real-time reporting).

## 🚀 Key Features
* **Smart Auth:** Automatically captures dynamic security tokens from network logs.
* **Audit Logic:** Merges raw stocktake data with tolerance tables to identify discrepancies.
* **Financial Accuracy:** Calculates exact penalty values and inventory gaps using CFA-aligned logic.
* **Auto-Sync:** Clears and overwrites old reports in Google Sheets for instant management updates.

## 📁 Structure
* **`scripts/`**: Core Python engine and automation files.
* **`data/`**: Automated export of timestamped audit trails.
* **`presentation/`**: Case study detailing ETL workflow and BI visualization.

## 📊 Impact
* **Efficiency**: 100% automated "one-click" execution.
* **Integrity**: Zero human error in complex penalty and unit price calculations.
* **Scalability**: Ready for multi-branch operation with minimal setup.
